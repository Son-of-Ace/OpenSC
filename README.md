---

## Roles
- Backend Devs should go read Documentation/backend
- Frontend Devs should go read Documentation/frontend
- UI Creators should go read Documentation/ui

## Backstory
What originally started as a passion project by me and a few others has now turned into something I wanted to share and make opensource so anyone can contribute. The original (ScriptCompanion1) had some pretty bad UI, but the idea was there and it was where it all started. OpenSC aka, ScriptCompanion2 was originally meant to be Private, but here we are.

## Goal
The goal with making ScriptCompanion opensource is that different people can come in with ideas and some people can learn something from the code or just look at it for the fun of it.
