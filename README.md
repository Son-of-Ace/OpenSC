OpenSC is a plugin made for Roblox Studio to help with tedious scripting or other tasks!
---

## Quick Start
- Backend Devs should go read Documentation/backend-dev/contribute-docs.rst
- Frontend Devs should go read Documentation/frontend-dev/contribute-docs.rst
- UI Designers should go read Documentation/ui-designer/contribute-docs.rst

## Roles
- Backend Devs job is to make systems or tools that the frontend developers can easily use. These systems and tools should be cleanly organized, well documented and easy to use. Your job is also to optimize code that needs it.

- Frontend Devs job is to make the code for the things the users interact with like UI. The UI for OpenSC is all made in React-Lua.

- UI Designers job is to make the UI for the Plugin. The style we are going for is kind of like VisualStudioCode. Try the plugin out before starting on making UI as it will give a better idea of the feel and style.

More info is included in the Documentation for each role.

## Backstory
What originally started as a passion project by me and a few others has now turned into something I wanted to share and make opensource so anyone can contribute. The original (ScriptCompanion1) had some pretty bad UI, but the idea was there and it was where it all started. OpenSC aka, ScriptCompanion2 was originally meant to be Private, but here we are.

## Goal
The goal with making ScriptCompanion opensource is that different people can come in with ideas and some people can learn something from the code or just look at it for the fun of it.

## Extra Info
The code is heavily under documented as of right now since it was meant to be a private project. I do however hope to improve this over time with the help of others too as the idea is to keep it a nice place for everyone trying to contribute.

There isn't much functionality in the plugin yet other than interface due to it being started recently and me being pretty busy with other things. There will be a more in-depth Document on the idea later.
